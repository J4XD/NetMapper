## ABOUT TOOL :

NetMapper is a lightweight Python script designed for IP lookup and gathering details about a specific target IP address. It works seamlessly on both rooted and non-rooted Android devices.


## AVAILABLE ON :

* Termux

### TESTED ON :

### REQUIREMENTS :
* internet
* storage 400 MB
* python
